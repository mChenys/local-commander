## 一、产品概述

### 1.1 产品定位

Local Commander Desktop 是一款**本地 AI 编程助手桌面应用**，通过调用本地 MLX 模型实现代码生成、图像分析、UI 自动化测试等功能，帮助开发者节省 90%+ Token 成本，同时保护代码隐私。

### 1.2 目标用户

| 用户群体 | 痛点 | 解决方案 |
|---------|------|---------|
| 个人开发者 | API 费用高、隐私担忧 | 本地模型、零费用 |
| 独立工作室 | 成本敏感、需多功能合一 | 一站式 AI 工具 |
| 企业团队 | 代码安全、合规要求 | 私有部署、数据不出本地 |
| 移动开发者 | UI 测试繁琐 | Android 自动化 |

### 1.3 核心价值

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   💰 成本优势: 本地模型运行，节省 90%+ API 费用              │
│                                                              │
│   🔒 隐私安全: 代码不离开本地，无数据泄露风险                │
│                                                              │
│   🚀 功能丰富: 代码生成 + 图像分析 + UI 自动化 + 知识库      │
│                                                              │
│   🎯 专业定位: 面向开发者，深度集成开发工作流                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 二、技术架构

### 2.1 技术选型

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| **前端框架** | React 18 + TypeScript | 组件化开发，类型安全 |
| **桌面框架** | Tauri 2.0 | 轻量级(~10MB)，原生性能 |
| **UI 组件** | Ant Design 5.x | 企业级组件库 |
| **状态管理** | Zustand | 轻量、简洁 |
| **编辑器** | Monaco Editor | VS Code 同款编辑器 |
| **后端服务** | Python + FastAPI | 复用现有 MCP 服务 |
| **进程通信** | Tauri IPC | 前后端通信 |

## 四、数据设计

### 4.1 数据模型

```typescript
// 对话
interface Conversation {
  id: string;
  title: string;
  model: 'coder' | 'vl' | '27b' | '7b';
  messages: Message[];
  createdAt: Date;
  updatedAt: Date;
}

// 消息
interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  model?: string;
  tokens?: number;
  createdAt: Date;
}

// Android 测试用例
interface TestCase {
  id: string;
  name: string;
  description: string;
  steps: TestStep[];
  createdAt: Date;
}

// 测试步骤
interface TestStep {
  action: 'tap' | 'swipe' | 'input' | 'back' | 'screenshot';
  element?: {
    text?: string;
    resourceId?: string;
    contentDesc?: string;
  };
  coords?: { x: number; y: number };
  input?: string;
  expected?: string;
}

// 知识点
interface Knowledge {
  id: string;
  text: string;
  category: string;
  tags: string[];
  embedding?: number[];
  createdAt: Date;
}

// 设置
interface Settings {
  theme: 'light' | 'dark' | 'system';
  language: string;
  defaultModel: string;
  autoSave: boolean;
  models: ModelStatus[];
}

// 模型状态
interface ModelStatus {
  name: string;
  downloaded: boolean;
  size: number;
  path: string;
}
```

## 八、风险与对策

| 风险 | 影响 | 概率 | 对策 |
|------|------|------|------|
| 模型下载慢 | 用户体验差 | 高 | 提供镜像源、预打包模型 |
| 内存不足 | 应用崩溃 | 中 | 模型按需加载、内存监控 |

| 跨平台问题 | 部分平台不可用 | 中 | 优先 macOS，逐步扩展 |
